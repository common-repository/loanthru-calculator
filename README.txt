== Description ==

LoanThru Calculator is a free and user-friendly loan calculator that allows you to select the interest calculation method to estimate the total loan interest and monthly repayment amount. Simply insert the shortcode [loanthru_calculator] to any posts or pages to display the calculator; or activate the LoanThru Calculator Widget on the sidebar.

= Major features of LoanThru Calculator include: =

* Editable Calculator Label (e.g., Home Loan, Car Loan, Personal Loan, etc...)
* Set a default value for Loan Amount
* Set a default value for Loan Tenure
* Set a default value for Interest Rate
* Set a default value for Interest Calculation Method
* Hide / Display Interest Calculation Method

== Installation ==

1. Upload the LoanThru Calculator plugin to your site and activate it.
2. Update the LoanThru Calculator settings located under Wordpress Settings.
3. Simply insert the shortcode [loanthru_calculator] to any posts or pages to display the calculator.
4. To activate LoanThru Calculator widget, simply drag the widget to a sidebar (located under Appearance > Widgets).


== Frequently Asked Questions ==

= What is the interest calculation method for Home Loan, Car Loan and Personal Loan =

* Monthly Reducing is the most commonly used interest calculation method for Home Loan. 
* Flat Rate is the most commonly used interest calculation method for Car Loan and Personal Loan.
